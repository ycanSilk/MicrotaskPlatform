## 二、快捷操作顶栏

### 结构组成
```
┌─────────────────────────────────────────────────────────┐
│ [快速发布] [余额: ¥1,234] [今日消耗: ¥567] [消息3] [头像] │
└─────────────────────────────────────────────────────────┘
```

### 设计规范
- 高度：72px
- 背景色：渐变#4A90E2 到 #6BA3F5
- 文字颜色：#FFFFFF
- 圆角：底部8px

### 核心功能
1. **快速发布按钮**
   - 大号按钮，突出显示
   - 点击直接进入任务创建流程

2. **实时余额显示**
   - 大字体显示当前可用余额
   - 点击跳转充值页面

3. **今日消耗统计**
   - 显示当日任务支出
   - 悬浮显示详细分类

## 六、移动端适配

### 手机端布局调整
```
┌─────────────────────────────────┐
│     💰 ¥1,234  📝  🔔3  👤      │ ← 顶部快捷栏
├─────────────────────────────────┤
│                                 │
│           主要内容区             │
│                                 │
│         (垂直堆叠卡片)           │
│                                 │
├─────────────────────────────────┤
│  🏠   📝   ✅   💰   📊        │ ← 底部导航
└─────────────────────────────────┘
```

### 移动端特殊功能
1. **快速拍照审核**
   - 支持相机直接拍摄截图
   - 图片自动上传和压缩

2. **语音备注**
   - 支持语音转文字备注
   - 快速记录审核意见

3. **推送通知**
   - 新任务提醒
   - 审核结果通知
   - 余额不足警告
