## 二、代理工作台顶栏

### 结构组成
```
┌─────────────────────────────────────────────────────────┐
│ [团队等级: 金牌代理] [佣金: ¥2,856] [下级: 23人] [👤]     │
└─────────────────────────────────────────────────────────┘
```

### 设计规范
- 高度：68px
- 背景色：渐变#FF9500 到 #FFB366 (橙色系)
- 文字颜色：#FFFFFF
- 金牌徽章图标

### 核心信息
1. **代理等级显示**
   - 等级徽章：铜牌/银牌/金牌/钻石
   - 升级进度条

2. **佣金收益**
   - 今日佣金收入
   - 点击查看详细报表

3. **团队规模**
   - 直属下级数量
   - 团队总人数
