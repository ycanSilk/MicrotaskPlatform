# 响应式布局设计方案

## 一、断点系统设计

### 设备断点定义
```css
/* 移动设备 */
@media (max-width: 767px) {
  /* 手机端样式 */
}

/* 平板设备 */
@media (min-width: 768px) and (max-width: 1023px) {
  /* 平板端样式 */
}

/* 桌面设备 */
@media (min-width: 1024px) and (max-width: 1439px) {
  /* 小桌面样式 */
}

/* 大屏桌面 */
@media (min-width: 1440px) {
  /* 大桌面样式 */
}
```

### 视口尺寸适配
```
移动端：320px - 767px
  - iPhone SE: 375px
  - iPhone 12: 390px
  - iPhone 12 Pro Max: 428px

平板端：768px - 1023px
  - iPad: 768px
  - iPad Pro: 1024px

桌面端：1024px+
  - 笔记本: 1366px
  - 台式机: 1920px
  - 4K显示器: 2560px+
```

## 六、性能优化策略

### 1. 图片懒加载
```javascript
// 移动端图片懒加载
const imageObserver = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const img = entry.target;
      const src = img.dataset.src;
      
      // 根据设备选择合适尺寸
      if (window.innerWidth <= 767) {
        img.src = src.replace('_large', '_mobile');
      } else {
        img.src = src;
      }
      
      imageObserver.unobserve(img);
    }
  });
});
```

### 2. 组件按需加载
```javascript
// 移动端简化版组件
const loadComponent = async (device) => {
  if (device === 'mobile') {
    return import('./components/MobileTaskCard');
  } else if (device === 'tablet') {
    return import('./components/TabletTaskCard');
  } else {
    return import('./components/DesktopTaskCard');
  }
};
```
