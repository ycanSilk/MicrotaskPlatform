# 评论员H5移动端布局设计

## 一、H5移动端优先设计

### 核心设计原则
- 主要使用终端：手机端 (iOS/Android)
- 基准屏幕：375px (iPhone 设计稿)
- rem单位系统：1rem = 16px
- 单手操作优先，拇指可达区域

### 整体布局结构
```html
<div className="h-screen flex flex-col bg-gray-50">
  <!-- 顶部收益栏 -->
  <header className="h-14 bg-gradient-to-r from-primary to-blue-500 text-white">
    <!-- 收益、抓单状态、通知 -->
  </header>
  
  <!-- 主内容区 -->
  <main className="flex-1 overflow-hidden">
    <!-- 任务大厅或其他页面 -->
  </main>
  
  <!-- 底部导航 -->
  <nav className="h-15 bg-white border-t border-gray-200">
    <!-- 4个主要功能导航 -->
  </nav>
</div>
```

## 三、任务大厅布局

### 任务卡片设计
```tsx
const TaskCard = ({ task }) => {
  return (
    <div className="bg-white rounded-lg p-4 mb-3 shadow-sm border border-gray-100">
      {/* 任务信息 */}
      <div className="flex items-start justify-between mb-3">
        <div className="flex-1">
          <h3 className="text-base font-medium text-gray-900 mb-1">{task.title}</h3>
          <div className="flex items-center space-x-4 text-sm text-gray-600">
            <span className="text-primary font-bold text-lg">￥{task.price}</span>
            <span>剩余 {task.remaining}/{task.total}</span>
            <span className="text-xs bg-gray-100 px-2 py-1 rounded">{task.category}</span>
          </div>
        </div>
        {task.isHot && <span className="text-xs bg-red-500 text-white px-2 py-1 rounded">🔥热门</span>}
      </div>
      
      {/* 任务要求 */}
      <div className="text-sm text-gray-600 mb-3">
        要求：{task.requirements.join(' + ')}
      </div>
      
      {/* 操作按钮 */}
      <div className="flex space-x-3">
        {task.countdown ? (
          <button className="flex-1 bg-primary text-white py-3 rounded-lg font-medium relative overflow-hidden">
            <span className="relative z-10">抓单 ({task.countdown}s)</span>
            <div 
              className="absolute top-0 left-0 h-full bg-white/20 transition-all"
              style={{ width: `${(10-task.countdown)/10*100}%` }}
            />
          </button>
        ) : task.remaining > 0 ? (
          <button className="flex-1 bg-primary text-white py-3 rounded-lg font-medium">
            立即抓单
          </button>
        ) : (
          <button className="flex-1 bg-gray-300 text-gray-500 py-3 rounded-lg font-medium" disabled>
            已抓完
          </button>
        )}
        <button className="px-4 py-3 border border-gray-300 rounded-lg text-gray-700">
          详情
        </button>
      </div>
    </div>
  );
};
```

## 四、底部导航设计

### 主导航结构
```tsx
const BottomNavigation = () => {
  const navItems = [
    { icon: '🏠', label: '大厅', path: '/hall' },
    { icon: '📝', label: '任务', path: '/tasks' },
    { icon: '💰', label: '收益', path: '/earnings' },
    { icon: '👤', label: '我的', path: '/profile' }
  ];
  
  return (
    <nav className="fixed bottom-0 left-0 right-0 bg-white border-t border-gray-200 px-2 py-2 safe-area-bottom">
      <div className="flex items-center justify-around">
        {navItems.map((item) => (
          <Link 
            key={item.path}
            href={item.path}
            className={`flex flex-col items-center justify-center min-w-0 py-2 px-3 rounded-lg transition-colors ${
              isActive(item.path) 
                ? 'text-primary bg-primary/10' 
                : 'text-gray-600'
            }`}
          >
            <span className="text-lg mb-1">{item.icon}</span>
            <span className="text-xs font-medium">{item.label}</span>
          </Link>
        ))}
      </div>
    </nav>
  );
};
```
